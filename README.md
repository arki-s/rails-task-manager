# 30days task manager

This app can manage the tasks you want to do continuously for 30 days.
Perfect for when you want to do a squat challenge, for example.

App home: https://WHATEVER.herokuapp.com


## Getting Started
### Setup

Install gems
```
bundle install
```
Install JS packages
```
yarn install
```

### DB Setup
```
rails db:create
rails db:migrate
rails db:seed
```

### Run a server
```
rails s
```
